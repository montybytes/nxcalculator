# NxCalculator

A simple calculator app that follows the design language of Nothing OS for Nothing and CMF phones.

## 💾 Download & Install
[<img height="80" alt="Get it on GitHub" src="./.github/assets/get-it-on-github.png" />](https://github.com/montybytes/nxcalculator/releases)

To test the app, find the `.apk` in [releases](https://github.com/montybytes/nxcalculator/releases) that matches your phone's architecture and download it or use `nxcalculator-release.apk` if you do not know the architechture.
Install from storage and (maybe) enjoy using the app.
Feedback is always welcome!

## 🎨 Design Philosophy

The calculator is designed to feel at home on Nothing & CMF phones. I have tried (to the best of my abilities) to follow Nothing OS' visual language to create a minimal, clean, and distraction-free UI.

Functionally, it is close to a 1:1 experience with Google's Calculator, while matching the Nothing ecosystem’s design principles.
